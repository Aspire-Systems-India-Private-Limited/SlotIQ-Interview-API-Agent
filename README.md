# SlotIQ Interview API Agent

This repository contains the SlotIQ Interview API Agent solution, including:

- Requirements and user stories
- Scaffold instructions and configuration
- .NET solution and projects for API, Logic, Data, Common, Integration, and UnitTests

## Structure

- `Requirement/` — Functional and technical requirements, user stories
- `Scaffold Instruction/` — Scripts and templates for scaffolding the solution
- `SlotIQ.Interview/` — Main .NET solution and projects

## Getting Started

1. Clone the repository
2. Follow the instructions in `Scaffold Instruction/` to scaffold or update the solution
3. Open `SlotIQ.Interview.sln` in Visual Studio or VS Code
4. Build and run the solution
